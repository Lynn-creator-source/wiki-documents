# Capture moving objects

## 概述

本示例介绍了如何利用**PIR传感器**实现对运动物体检测后唤醒抓拍功能。

### 技术背景

**PIR（被动红外）传感器**通过检测人体或动物发出的红外辐射变化来实现运动检测，具有以下特点：

- 检测范围：3-7米
- 检测角度：水平110°，垂直90°
- 工作电压：3.3V-5V
- 低功耗设计，适合电池供电场景

## 注意事项

1. **功耗管理**
   
   - 频繁触发会显著增加功耗
   - 电池供电场景建议：
     - 设置合理的触发间隔

2. **系统稳定性**
   
   - IDF v5.1.1存在深度睡眠唤醒BUG
   - 建议升级至v5.1.6版本

3. **抗干扰设计**
   
   - 远离电磁干扰源
   - 适当增加PULSE_C参数值

4. **环境适应性**
   
   - 避免直接阳光照射传感器
   - 保持传感器透镜清洁
   - 根据安装高度调整检测角度
